    在ubuntu登录界面隐藏git账户

    1. 首先跳转到以下目录

       cd /var/lib/AccountsService/users/

    2. ls后可以看到你想要隐藏的某个用户文件

    3. 然后打开该文件

       vim username

    4. 修改

       [User]
 XSession=
 SystemAccount=false  //将false改为true
