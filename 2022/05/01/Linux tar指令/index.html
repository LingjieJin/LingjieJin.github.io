    gzip

    иҜӯжі•пјҡgzip [йҖүйЎ№] еҺӢзј©пјҲи§ЈеҺӢзј©пјүзҡ„ж–Үд»¶еҗҚиҜҘе‘Ҫд»Өзҡ„еҗ„йҖүйЎ№еҗ«д№үеҰӮдёӢпјҡ

    -c е°Ҷиҫ“еҮәеҶҷеҲ°ж ҮеҮҶиҫ“еҮәдёҠпјҢе№¶дҝқз•ҷеҺҹжңүж–Үд»¶гҖӮ
    -d е°ҶеҺӢзј©ж–Үд»¶и§ЈеҺӢгҖӮ
    -l еҜ№жҜҸдёӘеҺӢзј©ж–Үд»¶пјҢжҳҫзӨәдёӢеҲ—еӯ—ж®өпјҡ
    еҺӢзј©ж–Үд»¶зҡ„еӨ§е°Ҹпјӣ
    жңӘеҺӢзј©ж–Үд»¶зҡ„еӨ§е°Ҹпјӣ
    еҺӢзј©жҜ”пјӣ
    жңӘеҺӢзј©ж–Үд»¶зҡ„еҗҚеӯ—
    -r йҖ’еҪ’ејҸең°жҹҘжүҫжҢҮе®ҡзӣ®еҪ•е№¶еҺӢзј©е…¶дёӯзҡ„жүҖжңүж–Үд»¶жҲ–иҖ…жҳҜи§ЈеҺӢзј©гҖӮ
    -t жөӢиҜ•пјҢжЈҖжҹҘеҺӢзј©ж–Үд»¶жҳҜеҗҰе®Ңж•ҙгҖӮ
    -v еҜ№жҜҸдёҖдёӘеҺӢзј©е’Ңи§ЈеҺӢзҡ„ж–Үд»¶пјҢжҳҫзӨәж–Үд»¶еҗҚе’ҢеҺӢзј©жҜ”гҖӮ
    -num з”ЁжҢҮе®ҡзҡ„ж•°еӯ— num и°ғж•ҙеҺӢзј©зҡ„йҖҹеәҰпјҢ-1 жҲ– вҖ“fast иЎЁзӨәжңҖеҝ«еҺӢзј©ж–№жі•пјҲдҪҺеҺӢзј©жҜ”пјүпјҢ-9 жҲ–вҖ“bestиЎЁзӨәжңҖж…ўеҺӢзј©ж–№жі•пјҲй«ҳеҺӢзј©жҜ”пјүгҖӮзі»з»ҹзјәзңҒеҖјдёә 6гҖӮ

    жҢҮд»Өе®һдҫӢпјҡ
    gzip *% жҠҠеҪ“еүҚзӣ®еҪ•дёӢзҡ„жҜҸдёӘж–Үд»¶еҺӢзј©жҲҗ .gz ж–Үд»¶гҖӮ
    gzip -dv *% жҠҠеҪ“еүҚзӣ®еҪ•дёӢжҜҸдёӘеҺӢзј©зҡ„ж–Үд»¶и§ЈеҺӢпјҢе№¶еҲ—еҮәиҜҰз»Ҷзҡ„дҝЎжҒҜгҖӮ
    gzip -l *% иҜҰз»ҶжҳҫзӨәдҫӢ1дёӯжҜҸдёӘеҺӢзј©зҡ„ж–Үд»¶зҡ„дҝЎжҒҜпјҢе№¶дёҚи§ЈеҺӢгҖӮ
    gzip usr.tar% еҺӢзј© tar еӨҮд»Ҫж–Үд»¶ usr.tarпјҢжӯӨж—¶еҺӢзј©ж–Үд»¶зҡ„жү©еұ•еҗҚдёә.tar.gzгҖӮ

    жҖ»з»“

    *.tar з”Ё tar вҖ“xvf и§ЈеҺӢ
    *.gz з”Ё gzip -dжҲ–иҖ…gunzip и§ЈеҺӢ
    *.tar.gzе’Ң.tgz з”Ё tar вҖ“xzf и§ЈеҺӢ
    *.bz2 з”Ё bzip2 -dжҲ–иҖ…з”Ёbunzip2 и§ЈеҺӢ
    *.tar.bz2з”Ёtar вҖ“xjf и§ЈеҺӢ
    *.Z з”Ё uncompress и§ЈеҺӢ
    *.tar.Z з”Ёtar вҖ“xZf и§ЈеҺӢ
    *.rar з”Ё unrar eи§ЈеҺӢ
    *.zip з”Ё unzip и§ЈеҺӢ

    tar вҖ“czf jpg.tar.gz *.jpg е°Ҷзӣ®еҪ•йҮҢжүҖжңүjpgж–Үд»¶жү“еҢ…жҲҗjpg.tarеҗҺпјҢе№¶дё”е°Ҷе…¶з”ЁgzipеҺӢзј©пјҢз”ҹжҲҗдёҖдёӘgzipеҺӢзј©иҝҮзҡ„еҢ…пјҢе‘ҪеҗҚдёәjpg.tar.gz
    tar вҖ“cjf jpg.tar.bz2 *.jpg е°Ҷзӣ®еҪ•йҮҢжүҖжңүjpgж–Үд»¶жү“еҢ…жҲҗjpg.tarеҗҺпјҢе№¶дё”е°Ҷе…¶з”Ёbzip2еҺӢзј©пјҢз”ҹжҲҗдёҖдёӘbzip2еҺӢзј©иҝҮзҡ„еҢ…пјҢе‘ҪеҗҚдёәjpg.tar.bz2
    tar вҖ“cZf jpg.tar.Z *.jpg е°Ҷзӣ®еҪ•йҮҢжүҖжңүjpgж–Үд»¶жү“еҢ…жҲҗjpg.tarеҗҺпјҢе№¶дё”е°Ҷе…¶з”ЁcompressеҺӢзј©пјҢз”ҹжҲҗдёҖдёӘumcompressеҺӢзј©иҝҮзҡ„еҢ…пјҢе‘ҪеҗҚдёәjpg.tar.Z
    rar a jpg.rar *.jpg rarж јејҸзҡ„еҺӢзј©пјҢйңҖиҰҒе…ҲдёӢиҪҪrar for linux
    zip jpg.zip *.jpg zipж јејҸзҡ„еҺӢзј©пјҢйңҖиҰҒе…ҲдёӢиҪҪzip for linux
